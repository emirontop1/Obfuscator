🛡️ Luau Obfuscator

Bu sistem, GitHub Actions ile otomatik obfuscation yapar.

Nasıl Kullanılır?

  1. src/ klasörüne .lua dosyalarını ekleyin.
  2. Değişiklikleri push'layın veya Actions'ı manuel çalıştırın.
  3. Obfuscate edilmiş dosyalar obfuscated/ klasöründe oluşur.

Detaylar için GitHub reposunu ziyaret edin.
